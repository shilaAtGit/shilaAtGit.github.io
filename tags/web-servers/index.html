CFX生成WS客户端

Shila Jin
2014-02-01
技术学习
记录一下如何根据CFX生成web servers的client端代码 WSDL介绍 WSDL 可描述网络服务（Web Services） WSDL 指网络服务描述语言 (Web Services Description Language)。 WSDL 是一种使用 XML 编写的文档。这种文档可描述某个 Web service。它可规定服务的位置，以及此服务提供的操作（或方法）。 下载CFX 下载地址：http://www.apache.org/dyn/closer.cgi?path=/cxf/2.7.11/apache-cxf-2.7.11.zip 将CXF压缩包解压到本地目录 配置环境变量 要根据操作系统的不同来配置。 配置环境变量 在CLASSPATH %CXF_HOME%\lib 在PATH中添加 %CXF_HOME%\bin 验证环境变量 验证：命令行输入： wsdl2java -v 结果显示：wsdl2java -Apache CXF 2.7.11说明配置成功 wsdl2java工具使用 使用命令行输入： wsdl2java -frontend jaxws21 -p com.test.client -d D:\ -client -autoNameResolution http://192.168.0.118:8080/framework/ws/helloService?wsdl 说明: -p 包名 -d 文件生成的本地目录 http://192.168.0.118:8080/framework/ws/helloService?wsdl 发布的服务访问路径 另外，记录： 以下的方法我没有用过，记录参考，万一用到 使用本地wsdl文件生成代码 使用命令行输入：wsdl2java -p com.test.client -d D: -verbose “D:\helloService.xml” 说明：-p 包名 -d生成文件路径 -verbose “wsdl文件路径”
